# Project Annex

This repository serves as a **code and configuration annex** for the project. It contains PHP classes, Twig templates, and configuration files for each application layer. 

## Structure

- **Entity_Layer/**: PHP classes representing data entities.
- **Controller_Layer/**: PHP classes managing request handling and application logic.
- **Templates_Layer/**: Twig template files for front-end rendering.
- **Typeform_Configuration/**: PHP/config files related to Typeform integration.
- **Environment_Configuration/**: `.env` file containing environment variables.
- **Security_Configuration/**: `security.yaml` showing access control and security rules.
